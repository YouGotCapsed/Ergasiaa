2.Επιχειρηματικό Μοντέλο
Ένα πιθανό επιχειρηματικό μοντέλο είναι η παροχή υπηρεσιών “Managed Support” σε επιχειρήσεις που διαθέτουν WordPress ιστοσελίδες. Το εργαλείο μπορεί να χρησιμοποιείται για βασικούς ελέγχους, όπως η εμφάνιση των τελευταίων άρθρων και ο έλεγχος της γενικής κατάστασης του site. Στο μέλλον μπορεί να επεκταθεί με περισσότερες λειτουργίες, όπως ενημερώσεις, διαχείριση περιεχομένου, αντίγραφα ασφαλείας και έλεγχος ασφάλειας. Οι πελάτες δεν πληρώνουν για τον ίδιο τον κώδικα, αλλά για την υποστήριξη, την εγκατάσταση, τις αναβαθμίσεις και τη συνεχή συντήρηση του συστήματος. Με αυτόν τον τρόπο το λογισμικό παραμένει ανοικτό, αλλά παράλληλα δημιουργούνται έσοδα από τις υπηρεσίες που προσφέρονται.
